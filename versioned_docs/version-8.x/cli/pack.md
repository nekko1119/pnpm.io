Create a tarball from a package.

## Options

### --pack-destination \<dir\>

Directory in which `pnpm pack` will save tarballs. The default is the current working directory.

### --pack-gzip-level \<level\>

Added in: v8.3.0

Specifying custom compression level.

## Life Cycle Scripts

- `prepack`
- `prepare`
- `postpack`
